Calculadora de IMC
Calculadora de Índice de Massa Corporal com classificação visual dinâmica. O usuário informa peso e altura em centímetros e recebe o IMC calculado com feedback de cor indicando a faixa — abaixo do peso, peso ideal, sobrepeso ou obesidade.

Ver online
🛠 Tecnologias

HTML5
CSS3
JavaScript

⚙️ Funcionalidades

Cálculo de IMC com conversão automática de cm para metros
Classificação por faixa com feedback visual
Layout responsivo para mobile

📦 Como rodar localmente
Clone o repositório e abra o index.html no navegador.
